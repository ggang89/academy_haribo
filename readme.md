    젤리의 시대다. 입가심으로 껌 대신 젤리를 씹고, 건강을 위해 '젤리형 숙취해소제'를 마시고 '영양 젤리'를 삼킨다.    
     **변화한 간식 시장**의 강자인 젤리! 그 중심에 **하리보**가 있다.    
    국내 매출 1위, 시장점유율 1위, 유럽에서 가장 신뢰받는 브랜드 **하리보**의 특색을 살리는 홈페이지를 만들어본다.
